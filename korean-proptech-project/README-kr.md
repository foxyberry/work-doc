# Rebuild Analytics (리빌드 애널리틱스)

> **재개발·재건축 및 시행 사업 타당성 검토에 특화된 B2B 분석 SaaS.**
> 흩어진 데이터를 한 곳에서, 조건을 바꿀 때마다 즉시, 보고서까지 한 번에.

리빌드 애널리틱스는 **재개발·재건축 및 시행 사업 타당성 검토에 특화된 분석 SaaS**입니다.

서비스의 핵심은 **필지 단위 데이터 파이프라인**입니다. 흩어져 있던 토지·건물·실거래·정비구역 데이터를 한 곳에 모아 사업 검토에 필요한 형태로 정리하고, 자체 개발 분석 지표로 가공하여, 정비조합·시행사·투자자가 의사결정에 바로 활용할 수 있는 유의미한 데이터로 제공합니다.

---

## 왜 사업성 검토가 중요한가

정비사업은 한 건이 짧게는 수년, 길게는 십수 년에 걸쳐 진행됩니다. 시공사·건설사 단위에서는 조 단위 자본이 투입되며, 초기 사업성 판단의 정확도가 사업 전체 ROI에 직접적으로 반영됩니다.

검토는 1회로 끝나지 않습니다. 후보지 발굴, 조합 합의, 시공사 협상, 자금 조달 단계마다 조건이 변하고, 그때마다 사업성을 다시 계산해야 합니다. 단계별 검토를 수기로 처리하면 작업 시간이 길어질 뿐 아니라 데이터 누락·계산 오류 가능성도 함께 증가합니다.

리빌드 애널리틱스는 이 반복 검토를 자동화하여, **조건 변경에 대한 즉각적인 재계산**과 **단계 간 일관된 산출 형식**을 제공합니다.

---

## 판단 전에 새는 시간

정비·시행 사업의 첫 검토는 계산보다 재정리에 시간이 듭니다. **후보지를 좁히고, 자료를 다시 정리하고, 데이터를 일일이 찾아보는** 사이 판단 타이밍은 지나갑니다.

리빌드 애널리틱스는 이 세 가지 병목을 정면으로 풉니다.

---

### 가치가 잠긴 지역을 먼저 찾으세요

지도 위 지역 탐색 기능으로 개발 잠재력이 있는 지역, 가치가 잠겨 있는 지역을 즉시 확인할 수 있습니다. 자체 개발한 분석 지표로 시세만으로는 보이지 않는 저평가 후보지를 골라내고, 한눈에 들어오는 색상 레이어로 광역 단위 탐색까지 한 자리에서 끝냅니다.

### 의사결정 자료, 매번 다시 만들지 마세요

흩어진 정부 데이터와 분석 결과를 단 한 번의 클릭으로 Excel·PDF 보고서로 받습니다. 외부 자문에게 보내거나 내부 보고에 그대로 제출할 수 있는 형태로 즉시 출력되고, 조건을 바꿀 때마다 보고서도 함께 새로 만들어집니다. 자료를 다시 모으고 정리하는 작업이 보고서 생성에 포함되어, 같은 시간을 의사결정 검토에 더 쓸 수 있습니다.

### 첫 질문은 AI 상담에게 던지세요

"이 지역의 평균 노후도는 어떻게 되나요?", "비슷한 조건의 사업성은 어떻게 다른가요?" 같은 질문에 자연어로 답합니다. 선택한 지역의 컨텍스트를 대화 내내 유지하며, 데이터를 찾고 계산하고 해석하는 첫 단계를 짧게 가져갑니다.

## 이 제품이 맞는 팀

| 조직 | 어떻게 쓰는가 |
|------|------|
| **정비조합 추진위** | 조합원 보고와 의사결정 자료를 빠르게 작성, 자체 분석 근거까지 첨부 |
| **시행사 사업개발팀** | 광역에서 신규 후보지를 발굴하고, 사업 조건별 손익을 단시간에 비교 검토 |

| **컨설팅·자문 조직** | 외부 고객 제출용 보고서를 즉시 출력하고, 자체 분석 근거로 보강 |

→ 사업성 검토의 첫 단계를 줄여야 하는 모든 조직에 적합합니다.

## 주요 기능

리빌드 애널리틱스의 모든 기능은 **재개발·재건축 사업 타당성 검토의 흐름 그대로** 연결됩니다.
지도에서 후보지를 발견하고, 폴리곤으로 지역을 잡아, 통합 데이터와 정비구역 정보를 확인하고, 시나리오를 시뮬레이션하고, 보고서로 받아, AI에게 묻는 한 흐름이 같은 화면에서 끝납니다.

### 1. 지도 기반 가치 발굴 (MapLayer) — *후보지 1차 스크리닝*

광역에서 후보지를 발굴하는 진입점입니다. 줌 레벨에 따라 시·구·동·필지 4단계로 사전 집계된 통계 지표가 색상으로 표현되어, 별도 분석 없이도 "어디부터 봐야 하는가"가 한눈에 들어옵니다.

| 기능 | 상세 |
|------|------|
| 4단계 줌 레벨 통계 | 시 · 구 · 동 · 필지 단위로 사전 집계된 지표를 즉시 표시 |
| 자체 분석 지표 표출 | Discovery Score (가치 발굴 지수), Land Leverage Score (재개발 잠재력) 등 |
| 색상 기반 탐색 | 점수 구간별 색칠로 광역에서도 후보지를 한눈에 식별 |
| 점수 필터링 | 임계값을 설정해 관심 점수대만 강조 표시 |
| 폴리곤 검색 연동 | 발견한 지역을 클릭 한 번으로 상세 분석에 그대로 진입 |

→ 지도에서 잡은 후보지가 다음 단계 상세 분석으로 끊김 없이 이어집니다.

### 2. 통합 토지·건물 데이터 조회 — *후보지 정밀 검토*

지도 위에 폴리곤을 그리거나 행정구역·정비구역을 그대로 선택하면, 그 안의 모든 토지·건물 데이터가 정합성을 맞춰 한 화면에 정리됩니다.

| 항목 | 상세 |
|------|------|
| 영역 입력 방식 | 자유 폴리곤, 행정구역, 정비구역, 지구단위계획구역 |
| 토지 데이터 | 공시지가, 용도지역, 면적, 소유 구분, 지목·형상 |
| 건물 데이터 | 구조, 층수, 사용승인일, 연면적, 노후 등급 |
| 정합성 보장 | 출처가 다른 정부 공공데이터 10종 이상을 자체 ETL로 통합. 한 필지에 대한 토지·건물·실거래·노후 정보가 일관된 식별자로 연결되어, 데이터 간 불일치나 누락 없이 조회됨 |

→ 출처별로 따로 자료를 찾아 맞추는 작업이 자동화되어, 검토 첫 라운드가 그만큼 짧아집니다.

### 3. 정비구역·도시계획 데이터 통합 — *사업 충돌·인접성 점검*

이미 진행 중이거나 의제 처리된 정비구역과 겹치는지, 인접한 사업이 있는지를 별도 조회 없이 같은 화면에서 확인합니다.

- 서울시 정비구역 (재개발·재건축 등 의제 처리분)
- 재정비촉진지구
- 도시개발구역
- 지구단위계획구역

→ 후보지가 다른 사업과 충돌하는지, 인접 사업으로 인한 가치 변화가 예상되는지 즉시 판단할 수 있습니다.

### 4. 사업 타당성 시나리오 시뮬레이션 — *사업성 계산·조건 재검토*

선택한 영역에 대해 핵심 조건을 조합하여 시나리오별 손익을 즉시 산출합니다. 한 번 영역을 잡으면, 조건만 바꿔도 결과가 다시 만들어집니다.

| 항목 | 상세 |
|------|------|
| 입력 파라미터 | 용적률, 평당 분양가, 평당 공사비, 사업 기간, 분담금 비율 등 |
| 출력 지표 | 개발 수익, 총사업비, 순수익, 비례율, 평당 수익률 |
| 토지 조서 자동 생성 | 선택 영역 내 모든 필지의 소유·면적·용도지역·노후도 정리 |
| 즉시 반영 | 조건 변경 시 시뮬레이션·토지 조서·보고서가 동시에 갱신 |

→ 조건만 바꾸면 결과가 다시 만들어지므로, 한 시나리오를 만드는 노력으로 여러 조건을 빠르게 검토할 수 있습니다.

### 5. Excel · PDF 보고서 다운로드 — *내부 보고·외부 자문 전달*

분석 결과를 클릭 한 번으로 받아 외부 자문에 보내거나 내부 보고에 그대로 제출할 수 있습니다.

**Excel 보고서 (4시트 구성)**

| 시트 | 포함 내용 |
|------|----------|
| 토지 요약 | 필지수, 면적, 용적률, 노후화율 등 종합 정보 |
| 건물 정보 | 건물별 상세 (구조·층수·사용승인일 등) |
| 사업성 분석 | 개발 수익·총사업비·순수익·비례율 시뮬레이션 결과 |
| 노후도 조건 | 노후 건축물 현황 및 판정 결과 |

→ 의사결정자에게 그대로 제출하거나, 추가 시뮬레이션을 위해 외부에서 재가공할 수 있는 형태로 제공됩니다.

**PDF 보고서**

- HTML 템플릿 기반 고품질 문서
- 차트·그래프 자동 삽입 (파이차트·바차트·라인차트)
- QR 코드로 원본 분석 페이지 접근 가능
- 외부 공유·인쇄 배포에 최적화

→ 조건이 바뀔 때마다 보고서도 함께 다시 만들어지므로, 보고용 자료를 별도로 재작성하는 작업이 사라집니다.

### 6. AI 부동산 상담 — *질의 응답·검토 가속*

선택한 지역의 데이터를 자연어로 질의해 답을 받습니다. 데이터 메뉴를 찾아 들어가지 않고도 묻고 바로 결과를 받는 인터페이스로, 첫 검토와 후속 검토 모두에서 활용됩니다.

**할 수 있는 일 예시**

| 질문 유형 | 예시 |
|------|------|
| 통계·요약 | "이 지역 평균 노후도는 어떻게 되나요?", "총 필지 수와 면적은?" |
| 사업성 해석 | "이 조건으로 진행하면 비례율이 얼마나 나오나요?" |
| 후보지 추천 | "비슷한 사업성을 가진 다른 후보지를 추천해주세요" |
| 다음 행동 안내 | "방금 분석 결과를 Excel로 받고 싶어요" |

**대화 특징**

- 선택한 지역의 컨텍스트가 대화 내내 유지됨
- 응답이 한 글자씩 스트리밍되어 빠른 체감 속도
- 대화 맥락에 맞는 후속 질문을 자동 추천하여 분석 깊이를 확장
- 외부 클라이언트(MCP 호환)에서도 동일한 분석 데이터에 접근 가능

→ 새로운 데이터 메뉴를 찾는 시간 없이, 묻는 즉시 답을 받습니다.

### 7. 작업 이어가기 (검색 히스토리·즐겨찾기·권한 관리) — *후속 검토 사이클 운영*

한 번 검토한 작업을 언제든 같은 조건으로 다시 열어보고, 관심 지역은 모아 관리합니다.

| 기능 | 상세 |
|------|------|
| 검색 히스토리 | 사용자별 검색 이력 자동 저장, 조건 그대로 재조회 |
| 즐겨찾기 | 관심 지역 저장·빠른 접근 |
| 사용 통계 | 일별·기간별 서비스 사용량 집계 (관리자 대시보드) |
| 권한 관리 | 게스트·일반·기업·관리자 4단계 RBAC |

→ 한 번 정리한 분석을 다음 검토 사이클에서 그대로 이어 받아 작업할 수 있습니다.

## 자체 개발 분석 모델

리빌드 애널리틱스의 분석 지표는 **재개발·재건축 가능성과 잠재력을 정량적으로 평가하기 위해** 자체 개발한 모델입니다. 단순 시세 조회에서 한 단계 나아간 가치 평가를 제공합니다.

### 가치 발굴 지수 (Discovery Score)

**무엇을 푸나** — 시세 비교만으로는 보이지 않는 잠재력 있는 후보지를 자동으로 골라냅니다.

**어떤 신호를 보나** — 가격·건물·지역 등 다중 데이터를 종합하여 "이 필지가 다음 사업의 후보가 될 만한가"를 정량 점수로 환산합니다.

**실무 활용**
- **광역 후보지 발굴**: 지도 위 점수 높은 지역을 한눈에 식별
- **정량 비교 기준**: 여러 후보지를 동일 척도로 객관적 비교
- **외부 보고 근거**: 점수와 함께 의사결정 자료로 활용

### 재개발 잠재력 지표 (Land Leverage Score)

**무엇을 푸나** — 토지 가치 대비 건물이 약한, 즉 재개발 옵션의 가치가 큰 필지를 식별합니다.

**어떤 신호를 보나** — 부동산 경제학의 **Land Leverage** 개념(Munneke & Womack, *Real Estate Economics* 48(1), 2020)을 한국 공시 데이터에 맞춰 구현한 자체 점수 모델입니다.

**실무 활용**
- **재개발 후보지 사전 발굴**: 시장이 아직 합의하기 전에 후보지를 먼저 식별
- **Discovery Score와 교차 검증**: 두 지표가 동시에 강한 신호를 보내는 필지를 우선 검토
- **외부 보고 근거**: 학술적으로 검증된 개념 기반의 정량 지표로 활용

### 지역별 시세 보정 모델

공시가격과 실제 시장 가격 사이의 간극을 한국부동산원의 아파트 매매가격지수를 활용해 보정하는 모델입니다.

**도입 배경:**
- 공시가격은 연 1회 산정되어 급변하는 시장 상황을 반영하지 못함
- 지역별 가격 변동률 격차가 큼 (서초구 +17.3%, 도봉구 -14.0% 같은 극명한 격차)

**모델 특징:**
- 공시가격 이후의 순수 시장 변동분만 보정
- 장기 누적 효과로 인한 가중치 과대 산정 방지
- 지역별 고유의 시장 흐름 반영

### 시계열 데이터 정제

연도별 평당 분양가, 실거래가 등 시계열 데이터에 흔히 발생하는 누락·이상치를 자동 처리하여 **분석과 차트에서 일관된 흐름**을 보장합니다.

**B2B 활용**
- **추세 분석**: 데이터 결손으로 인한 잘못된 해석 방지
- **차트 시각화**: 외부 보고서·발표 자료에서 끊김 없는 시계열 제공
- **시뮬레이션 안정성**: 비정상 값으로 인한 시뮬레이션 왜곡 방지

→ 정부·시장 데이터의 누락과 튀는 값을 자동으로 정리해, 사용자가 데이터 신뢰성을 별도 검증하지 않아도 되도록 합니다.

### 행정 단위별 통계 지표

전국 데이터를 시·구·동·필지 단위로 정리하여, 광역 시장 추세부터 개별 필지 검토까지 같은 화면에서 끊김 없이 비교할 수 있도록 제공합니다.

**제공 지표**
- 평균 공시지가, 시장가-공시가 괴리도
- 추정 토지가치, 자산 밀도
- 거래량·평균 분양가 등 시장 활성도 지표
- Discovery Score, Land Leverage Score 등 자체 발굴 지수

**B2B 활용**
- **광역 모니터링**: 시·구 단위 추세 파악
- **후보지 발굴**: 동 단위 비교로 관심 지역 좁히기
- **개별 검토**: 필지 단위에서 사업성 시뮬레이션으로 연결

→ 같은 지표를 단위만 바꿔 보면서, 광역 흐름과 개별 필지를 한 흐름으로 분석할 수 있습니다.

## 제공 데이터 · API

리빌드 애널리틱스가 다루는 데이터·기능 범위는 다음과 같습니다. 일부 기본 기능은 별도 절차 없이 사용 가능하며, 분석 지표와 정밀 시장 데이터는 **기업 협약 또는 로그인 사용자 대상**으로 제공됩니다.

### 기본 제공 기능

- **지오 검색**: 주소·필지·건물 검색, 통합 검색
- **필지 기본 정보 조회**: 필지 단위 기본 정보 (가격·점수 등은 마스킹 처리)
- **실거래 요약**: 영역 내 평단가 등 통계 요약 (개별 거래는 비공개)
- **정비구역·도시계획 조회**: 정비구역·재정비촉진지구·도시개발구역
- **지역 요약 통계**: 행정구역 단위 요약 지표

### 기본 제공 데이터 필드

**필지 기본 정보**
- 고유 ID, 지번, 법정동명
- 면적, 지목, 용도지역 (1·2종)
- 건폐율, 용적률
- 필지 경계 (geometry)
- 건물 유형, 세대수, 가격 기준 연도

**건물 요약**
- 구조, 층수, 사용승인일, 노후 등급

**실거래 요약**
- 영역 내 거래량, 평균 평단가 등 통계 지표

### 기업 협약 / 로그인 사용자 전용 제공

아래 항목은 **자체 개발 분석 지표** 및 **정밀 시장 데이터**로, 기업 협약 또는 로그인 사용자에게만 제공됩니다.

- **자체 분석 지표**: Discovery Score, Land Leverage Score, 시장-공시 괴리도, 저평가 점수
- **지도 레이어 정밀 지표**: 공시지가 수치, 자산 밀도, 평균 건물 연식, 노후 건물 비율, 평균 분양가, 중앙 토지가치 (시·구·동 단위)
- **정밀 가격 데이터**: 추정 토지가치, 평당 분양가·전세가·전세가율
- **개별 실거래 내역**: 영역 내 모든 거래 내역 상세
- **사업성 시뮬레이션 결과**: 시나리오별 손익, 비례율 등
- **보고서 다운로드**: Excel·PDF 즉시 출력
- **저장·관리 기능**: 검색 히스토리, 즐겨찾기

→ 도입 형태 및 가격 정책은 [도입 방식](#도입-방식) 섹션 참고 또는 별도 문의 바랍니다.

## 도입 방식

리빌드 애널리틱스는 기본 웹 SaaS 외에도 **데이터·분석·AI 자산을 다양한 형태로 제공**합니다. 기업의 환경과 활용 시나리오에 맞춰 협약 단위로 도입 형태를 설계합니다.

| 도입 형태 | 적합한 조직 | 설명 |
|------|------|------|
| **웹 SaaS** | 정비조합·시행사·투자 검토팀·컨설팅 (모든 직접 사용자) | 표준 웹 인터페이스. 지도 탐색·시뮬레이션·보고서 다운로드까지 한 화면에서. |
| **API 라이선스** | 자체 시스템·플랫폼을 보유한 기업 | 기업 자체 시스템에 데이터·분석 결과를 직접 통합. **건당 호출 과금** 또는 **계약 단위(월·연)** 과금 가능. |
| **MCP 서버 제공** | AI·LLM을 적극 활용하는 조직 | Claude Desktop 등 외부 LLM 클라이언트에서 우리 데이터를 자연어로 조회. 협약 단위 라이선스. |
| **AI 상담봇 제공** | 자사 서비스에 부동산 상담을 임베드하려는 기업 | 자사 서비스에 부동산 상담 봇 임베드. 데이터·LLM·UI 통합 패키지 제공. |

**기업 협약 활용 예시**
- **분석 API 통합**: 기업 내부 의사결정 시스템에 우리 데이터·점수를 직접 연결
- **컨설팅 자동화**: 외부 API → 컨설팅사 보고서 템플릿 자동 생성
- **부동산 플랫폼 임베드**: 부동산 정보 서비스 안에 우리 점수·통계 노출
- **외부 챗봇 연동**: 기업 메신저·앱 안에서 부동산 데이터 자연어 응답

**협약 상담에서 함께 검토하는 항목**
- 데이터 범위 — 어떤 지역·어떤 지표를 얼마나 자주 받을 것인가
- API 제공 방식 — 건당 호출, 월·연 단위 계약, 전용 엔드포인트 등
- 사내 시스템 연동 — 기존 의사결정 시스템·BI 도구·CRM과의 통합 방식
- AI 상담봇 적용 범위 — 챗봇 임베드 위치, 응답 도메인, 사용자 권한

→ 사용 규모·활용 범위·운영 방식에 맞춰 협약 단위로 맞춤 설계합니다. 도입 논의가 필요한 경우 별도 안내드립니다.
